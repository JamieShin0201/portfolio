하루하루 성장하는 개발자

Yogiro (소모임 어플)

여러가지 분야의 소모임들을 만들고
모임내에서 게시판, 사진첩, 일정만들기 기능들을 이용할수 있는 어플입니다.

( 개발인원: 1명(혼자), 개발기간: 4주, 사용언어: JAVA )

1. 회원가입

1-1) 관심사 등록 : 1개이상의 관심사를 등록하도록 구현
1-2) 이메일, 비밀번호, 이름 정규식을 이용한 예외처리
1-3) 프로필사진 등록, DatePicker를 이용한 생년월일 등록

2. 로그인, 로그아웃, 아이디저장, 자동로그인

3. 프로필 변경, 관심사 변경

4. 모임 목록

4-1) RecyclerView를 활용하여 모임 목록을 구현
4-2) 카테고리별로 모임을 볼수 있도록 구현

5. 모임 가입 및 내가 가입한 목록,
내가 만든 목록, 내가 본 목록

6. 모임 만들기

6-1) 관심사, 지역, 모임이름과 내용을 입력받도록 구현
6-2) 공공데이터를 활용해 지역을 입력하도록 구현

7. 모임 탈퇴

8. 모임 내 게시판 글 및 댓글, 대댓글 작성

9. 모임 내 사진첩

9-1)ImagePicker를 이용해 다중사진 업로드가 가능하도록 구현

10. 소모임 내 정모 만들기

10-1)DatePicker와 TimePicker를 이용해 날짜와 시간을 입력하도록 구현
10-2)Naver static map API과 dynamic map API을 이용하여 지도를 입력하도록 구현

11. 핸들러를 이용한 이상한 가위바위보 게임
